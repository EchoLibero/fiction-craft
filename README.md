# Fiction Craft

Конспекты и анализ канонических произведений фантастики. Рабочий инструмент для писательства.

**Живой сайт:** https://echolibero.github.io/fiction-craft/

## Структура

- `index.html` — главная
- `genre-map.html` — карта жанра и канона
- `neuromancer-1.html` — первый конспект по Нейроманту

## Цель

Картировать жанр изнутри: знать сюжетные ходы, видеть затёртые паттерны, находить белые пятна.

Для сильной фантастики нужно знать что уже сделано — чтобы не повторяться и добавлять своё.
